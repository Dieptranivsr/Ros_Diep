
# Building Packages

### Building Your Package
```shell
$ cd ~/<...>/ros_ws/
$ rm -rf build
$ cd build
$ cmake ../src/ -DCMAKE_INSTALL_PREFIX=../install -DCATKIN_DEVEL_PREFIX=../devel
```
